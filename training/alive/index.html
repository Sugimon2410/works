アライブについて

全日制の英語の保育園（幼稚園の一体型）、英会話教室のイングリッシュスクール、英語でSDGsや
専門分野を学ぶアカデミー、英語で理工系を学ぶSTEM スクール、学童コース、オンラインコースがあります。

Courses

色々なコース

インターナショナル
スクール

英語の全日制の保育園（幼稚園の一体型）です。
週2日2時間～1歳・2歳コースあり

View More

イングリッシュ
スクール

園や学校の後に通っていただく、
習い事のコースです。
親子で一緒に受けていただくクラス
（プレップクラス）もあります。

View More

STEMスクール

英語で、理工系の分野を学ぶコースです。
クリティカル思考、問題解決、クリエイティビティが身につきます。

View More

ドリームスクール

英語で行う学童プログラムで14:10-19:30までの
利用が可能なコースです。
学校の宿題はもちろん、
英会話や塾の学習（日本語）もあり、
色々なことにチャレンジできます。

View More

アカデミーコース

知識だけでなく、
クリティカル思考、問題解決、
クリエイティビティが身につきます。

View More

シーズナルスクール

・スプリングスクール
・サマースクール　・STEMキャンプ
・リーダーシップアントレキャンプ
・プレナーキャンプ

View More

オンラインスクール

英語を学びたい方、英語で専門分野について
学びたい方を対象にした、
質の高いオンラインレッスンコースです。
